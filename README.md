# NaniCobo Secrets — Website Development

## Descripción

NaniCobo Secrets es un proyecto de desarrollo web enfocado en la creación de una experiencia ecommerce premium para una marca especializada en peluquería profesional y cuidado capilar. El objetivo principal de la web es construir una experiencia visual moderna, elegante y altamente inmersiva que combine branding, motion design y rendimiento técnico.

La plataforma está diseñada siguiendo una filosofía mobile-first y performance-first, priorizando tiempos de carga rápidos, animaciones fluidas y una experiencia visual cinematográfica. El proyecto busca alejarse de la estructura tradicional de ecommerce y acercarse más a una experiencia editorial inspirada en marcas luxury beauty, skincare premium y campañas visuales cinematográficas.

La web está siendo desarrollada con una arquitectura moderna basada en React y Next.js, utilizando componentes reutilizables, diseño responsive avanzado y un sistema visual consistente orientado a escalabilidad futura.

---

# Objetivos del Proyecto

El desarrollo de la web tiene varios objetivos técnicos y visuales:

- Crear una landing page altamente visual e inmersiva.
- Construir una experiencia premium enfocada en branding.
- Implementar animaciones scroll-driven fluidas.
- Integrar vídeo cinematográfico optimizado.
- Desarrollar una arquitectura reusable y escalable.
- Mantener un alto rendimiento incluso con contenido visual pesado.
- Optimizar la experiencia móvil.
- Preparar una futura integración ecommerce completa.

La intención es que la web funcione tanto como plataforma de marca como herramienta de conversión.

# Stack Tecnológico

## Frontend

El frontend está desarrollado utilizando tecnologías modernas del ecosistema React:

- Next.js
- React
- TypeScript
- TailwindCSS
- Framer Motion

Next.js se utiliza como framework principal debido a sus capacidades de:

- Server Side Rendering (SSR)
- Optimización automática de assets
- Routing moderno
- Optimización SEO
- Escalabilidad
- Gestión eficiente de imágenes y vídeo

TypeScript permite mantener un código más seguro, mantenible y escalable a largo plazo, especialmente en componentes complejos y sistemas de animación.

TailwindCSS se utiliza para construir una interfaz completamente custom manteniendo consistencia visual, velocidad de desarrollo y diseño responsive avanzado.

Framer Motion se utiliza para:

- Animaciones de entrada
- Scroll animations
- Motion transitions
- Element reveals
- Cinematic movement
- UI microinteractions
